# hardware-adder
this adder can be used in pocket digital calculators that holds 9 digits
36-Bit Synchronous Ripple Carry AdderA synthesizable, 36-bit synchronous ripple carry adder implemented in Verilog. This module is designed for digital hardware projects, such as pocket calculators or arithmetic logic units (ALUs), where high-width bit manipulation (e.g., 9-digit BCD or Hex operations) is required.
FeaturesSynchronous Design: Registers outputs on the positive edge of the clock.Active-High Reset: Synchronously clears outputs to a safe zero state.Continuous Monitoring: Provides a combined 37-bit real-time output (s1) containing the internal sum and final carry bit.36-Bit Precision: Ideal for processing 9-digit nibble-based operations (9 digits × 4 bits/digit).

Circuit Logic & ArchitectureThe module utilizes an internal for loop that mimics a chain of 36 full adders.Sum Generation: Evaluated sequentially using XOR gates:temp_sum[i] = a[i] ^ b[i] ^ c[i]Carry Generation: Evaluated using look-ahead routing emulated by the loop:c[i+1] = (a[i] & b[i]) | (c[i] & (a[i] ^ b[i]))Real-time Vector (s1): Continuously combines the 36th carry bit and temp_sum for immediate debugging or lookahead operations prior to the next clock edge.
